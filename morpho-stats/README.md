# Morphological statistics

This directory contains the code to reproduce the subword computations in the paper. There are 4 directories:

- all-morph: All morphological alignment computations including WER, alignment rates and all.
- avg-subwords-counts: Computation of average subwords over several corpora.
- fertility: Fertility and single token rate computations.
- sample-words-break: Small code to exhibit how a small list of words are broken by tokenizers.
