This directory runs experiments for character-level, word-level, morpho-subwords vocab types. For the character-level you can run the experiments directly. For word level and subword evel first one needs to prepare vocabulary files, please see the related subdirs. For each voxab size, we included vocab making code and resulting vocab and coverage files as well. Since the vocab files existent, one can run the experiments directly. The experiment code includes experimenting with several vocab sizes.
